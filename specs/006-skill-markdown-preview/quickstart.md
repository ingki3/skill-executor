# Quickstart: Skill Markdown Preview

This feature allows you to view rendered documentation for any registered skill.

## 1. Register a Skill
Register a skill from a repository that contains a `skill.md` or `SKILL.md` file (e.g., `https://github.com/ComposioHQ/awesome-claude-skills/tree/main/freshdesk-automation`).

## 2. Open Skill Management
Locate the newly registered skill in the "Skill Management" dashboard.

## 3. View Documentation
1. Click the **View Docs** button on the skill card.
2. The card will expand to show the rendered documentation.
3. Review the headers, lists, and code blocks to understand the skill's capabilities.

## 4. Collapse Docs
Click the **Hide Docs** button to return the card to its compact view.
